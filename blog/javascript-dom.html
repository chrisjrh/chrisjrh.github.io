Differences Between HTML and CSS

HTML: HTML covers all the text for a website and provides the links for anything like background pictures and visual exmaples, its basically all the core components of a website.

CSS: CSS orders everything and sorts out the space between paragraphs, colours the text and picks the area/location of photos.

Control Flow Loops

A control flow loop is typically executed from top to bottom and are used to go through the stages of completing a task. For starting a car it would look like:

  • Unlock car with key
  • Open door
  • Turn key in the ignition
  • Car starts

Explaining the DOM

DOM stands for: Document Objet Model, in basic terms it is the interface for web documents allowing you to change/adjust certain objects on a website and seeing them update in live time. You can use Shift-CTRL-I on google chrome to open it.

Differences between accessing data from Arrays and Objects

Objects are a very powerful tool in Javascript and they basically are used for almost anything. Objects can store data for anything inside of Javascript

Arrays are also Objects but the difference is that an Array stores data in an ordered fashion. In other words if you want to save data in sequential order you use and Array, if not you use an Object.

What are Functions

Functions are used to prevent repeating blocks of code by giving a set of instructions grouped together to calculate a specific outcome. You can call this function as many times as you want and only need the contents of the function to be written out once.
